학교안전사고 빅데이터 분석
데이터 추출 · 정리 · 보고서/PPT 제작

학교안전공제회 데이터를 기반으로 분석용 데이터셋을 구축하기 위해, 데이터 추출/정리(ETL 전 단계)와 보고서·PPT 팜플렛 제작을 담당했습니다.
분석(모델링/통계/인사이트 도출)은 별도 분석 담당자가 수행하고, 저는 분석이 가능한 형태로 데이터 품질과 구조를 맞추는 역할에 집중했습니다.

프로젝트 설명

본 프로젝트는 학교안전사고 관련 대규모 데이터를 기반으로 “사고 규모/특성/영향 요인”을 체계적으로 분석하기 위한 기반 작업을 포함합니다.

저는 데이터 엔지니어로서 분석팀이 바로 활용할 수 있도록 데이터 추출과 구조 정리를 담당했습니다. Navicat과 PostgreSQL을 사용해 원천 데이터를 추출하고, Excel을 통해 코드 표준화/결측 처리/테이블 정합성 점검 등 전처리(분석 전 단계)를 수행했습니다.

또한 학교안전공제회와의 소통 창구 역할을 맡아 데이터 요청/정의/검증 과정을 진행했고, 최종적으로 결과 보고서 작성 및 PPT 팜플렛 형태의 산출물을 제작하여 이해관계자가 전체 흐름(데이터 수집 → 전처리 → 분석 → 보고서)을 쉽게 이해할 수 있도록 구성했습니다.

담당 업무

  • 데이터 추출: PostgreSQL 기반 원천 데이터 조회/추출(Navicat), 분석 대상 기간/범위에 맞춘 데이터 확보
  • 데이터 정리: 코드 표준화, 컬럼 정의 정리, 결측/이상치 후보 확인, 테이블 간 키/정합성 점검
  • 분석용 데이터셋 구성: 분석팀 전달용 형태로 데이터 구조/변수 사전 정리(분석은 별도 담당)
  • 커뮤니케이션: 학교안전공제회 데이터 요청/확인, 정의 변경사항 반영 및 검증 지원
  • 산출물 제작: 최종 보고서 작성, PPT 팜플렛 제작(전달력 중심의 시각화/레이아웃 구성)
